% macros.tex --- shared notation and commands for the paper.
%
% One symbol, one definition, defined once here; sections never redefine it.
% Ordered general -> specific so a section can be pruned wholesale: a pure-maths
% paper drops "Machine learning"; a paper that doesn't need the random/
% deterministic typeface distinction drops "Random variables"; and so on.
%
% Requires (all loaded in preamble.tex): amsmath/mathtools
% (\DeclareMathOperator, \DeclarePairedDelimiter), amssymb (\mathbb), bm (\bm).
%
% Contents:
%   1. Basic mathematics        number sets, operators, delimiters, norms
%   2. Probability & statistics expectation, variance, divergences, distributions
%   3. Linear algebra           vectors, matrices, tensors, sets (deterministic)
%   4. Random variables         the random-typeface counterparts of section 3
%   5. Machine learning         data, model distributions, losses, activations
%   6. Document helpers         caption panels, term emphasis, spacing
%
% Convention (sections 3-4): vectors are bold lower-case, matrices bold
% upper-case, tensors bold sans-serif, sets blackboard, index sets calligraphic;
% random quantities repeat those shapes in upright roman.

% width rebinding as a result. In a single-column layout the two coincide, which

% definition instead of keeping its own copy.
